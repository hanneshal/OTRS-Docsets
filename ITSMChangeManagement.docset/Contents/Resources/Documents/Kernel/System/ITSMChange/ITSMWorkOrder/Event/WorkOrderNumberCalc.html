PRIVATE INTERFACE

_WorkOrderNumberCalc()

This method actually recalculates the WorkOrderNumbers for the workorders of the given change. It returns 1 on success, undef otherwise.

    my $Success = $EventObject->_WorkOrderNumberCalc(
        ChangeID => 123,
        UserID   => 1,
    );
